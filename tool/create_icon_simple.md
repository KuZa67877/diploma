# Быстрое создание иконки

Поскольку программная генерация PNG требует дополнительных зависимостей, 
вот простой способ создать иконку:

## Вариант 1: Онлайн генератор

1. Перейдите на https://icon.kitchen/
2. Выберите:
   - Shape: Squircle (закругленный квадрат)
   - Background: Gradient от #1F7A71 до #2A9D8F
   - Icon: Brain (или Psychology)
   - Padding: Medium
3. Скачайте все размеры для iOS и Android
4. Распакуйте в соответствующие папки проекта

## Вариант 2: Figma/Design Tool

1. Создайте квадрат 1024x1024
2. Примените градиент от #1F7A71 (верх-лево) до #2A9D8F (низ-право)
3. Закругление углов: 226px
4. Добавьте brain иконку (белый цвет, размер ~512px)
5. Экспортируйте как PNG

## Вариант 3: Использовать существующую иконку

Скопируйте иконку из дизайна и поместите в:
```
assets/images/app_icon.png (1024x1024)
```

Затем запустите:
```bash
flutter pub run flutter_launcher_icons
```
